# returncsgo

Сборник карт, фиксов и туториалов для запуска и оживления CS:GO.

Как скачать:
1. Справа на странице найдите блок **Releases**.
2. Нажмите на актуальный релиз (`Оживление CS GO`).
3. Скачайте прикрепленный **RAR-архив**.

Инструкция по установке:
1. Куда кидать карты
Распакуйте файлы карт (формат `.bsp`) в папку игры:
`...\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\maps`

3. Как запустить карту через консоль
1. Включите консоль в настройках игры («Параметры игры» -> «Включить консоль разработчика (~)» -> Да).
2. Нажмите клавишу ~ (Ё) на клавиатуре.
3. Введите команду: `map название_карты` (например, `map de_dust2`) и нажмите Enter.
