Сборник материалов для оживления старой доброй CS:GO! Здесь собраны карты, фиксы и полезные туториалы.

Как скачать всё необходимое?
Сам архив с картами весит 27 МБ, поэтому он выложен в специальный раздел.

1. Посмотрите в правую колонку этой страницы.
2. Найдите блок **Releases** (Релизы).
3. Нажмите на самый свежий релиз и скачайте прикрепленный **RAR-архив**.
